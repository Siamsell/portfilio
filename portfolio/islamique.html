Project Overview

The Islamic Ramadan Portal was designed to provide Muslims worldwide with essential tools and resources during the holy month. The platform combines spiritual guidance with practical functionality to enhance the Ramadan experience.

As the lead developer, I created a responsive web application that adapts to all devices, ensuring accessibility for users whether at home or on the go during Taraweeh prayers.

Technologies Used

HTML5 Tailwind CSS Java Script Firebase

Key Features

Prayer Times

Accurate prayer times based on geolocation with Imsak and Iftar alerts

Quran Reader

Complete Quran with translation and audio recitation

Zakat Calculator

Interactive tool to calculate and distribute Zakat al-Fitr

Iftar Finder

Community-powered map of local Iftar events and free meals

Mosque Locator

Find nearby mosques with Taraweeh prayer schedules

Progress Tracker

Track fasting days, Quran reading progress and good deeds
